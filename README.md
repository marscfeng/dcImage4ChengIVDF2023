# dcImage4ChengIVDF2023
MATLAB scripts to illustrate three frequently-used dispersion imaging techniques, MAPS, SPAC and ReMi used in Cheng et al., High-resolution near-surface imaging at the basin scale using dark fiber and distributed acoustic sensing: towards site effect estimation in urban environments. 2023
